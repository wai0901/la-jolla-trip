# La Jolla 之旅行程表

## 1. Pinpoint Cafe
這間咖啡廳位於 La Jolla Cove 附近，擁有極佳的視野，適合作為旅程的起點，享受一杯高品質的咖啡並欣賞海景。
[Google Maps: Pinpoint Cafe](https://www.google.com/maps/search/?api=1&query=Pinpoint+Cafe+La+Jolla)

## 2. Coast Walk Trail
La Jolla 最優美的沿海懸崖步道，沿途可俯瞰絕佳的加州海岸線，以及觀察在礁岩上休息的海獅。
[Google Maps: La Jolla Coast Walk Trail](https://www.google.com/maps/search/?api=1&query=La+Jolla+Coast+Walk+Trail)

## 3. Coffee Break (TBA)
在步道後，這是一個輕鬆的喘息時間，可以在 La Jolla Village 附近尋找這區具特色的獨立咖啡店。
[Google Maps: Coffee Near La Jolla Village](https://www.google.com/maps/search/?api=1&query=Coffee+Shop+near+La+Jolla+Village)

## 4. Top Gun House, Oceanside
電影《捍衛戰士》(Top Gun) 中著名的拍攝地點，重現了電影中的經典場景，也是 Oceanside 的熱門地標。
[Google Maps: Top Gun House](https://www.google.com/maps/search/?api=1&query=Top+Gun+House+Oceanside)

## 5. Truly Pizza, Dana Point
這間是在地備受好評的披薩店，以其酥脆餅皮與豐富配料聞名，作為這次旅程的完美結尾。
[Google Maps: Truly Pizza](https://www.google.com/maps/search/?api=1&query=Truly+Pizza+Dana+Point)
